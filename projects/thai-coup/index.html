Thai Coup Snapshot

On Thursday the Thai military staged a coup. Protest leaders were arrested and international TV networks were shut down as the army moved through the streets of Bangcock declaring martial law. However, at least as of early Friday morning, one of the more remarkable aspects of this unsettling takeover has been Thai citizen's relatively non-chalant response to the crisis. People are taking selfies and posing with soldiers, while Twitter users have adopted the hashtag #CoupMovies to make movie-title puns like "One Flew Over the Coup Coup's Nest."

We used the CrisisNET platform to quickly collect updates from over 100 Twitter and Instagram accounts, and generated a real-time snapshot of the mood in Thailand in a fraction of the time it would have taken normally. The feed is updated once per minute. Refresh the page to see the latest content, and scroll to the bottom of the page to see a simple code example you can use to put this data into your application.

Code Examples

Getting data from CrisisNET couldn't be easier. Below is a simple example in JavaScript showing how to retrieve the 100 most recent incidents about Thailand. We're currently in private alpha until June, but you can sign up for our mailing list and we'll let you know when it's public.

$.ajax({
  url: 'http://devapi.crisis.net/item?placeName=thailand&limit=100',
  dataType: "json",
  beforeSend: function(xhr) {
    xhr.setRequestHeader('Authorization', 'Bearer YOUR API KEY');
  },
  success: function (data) {
    console.log(data);
  }
});
